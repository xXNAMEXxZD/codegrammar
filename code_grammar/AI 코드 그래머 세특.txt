프로젝트명
AI 코드 그래머

목적
이 프로그램의 목적은 학생들이 작성한 Python 코드를 빠르게 검사하고, 문법 오류를 확인하며, 필요할 경우 ChatGPT를 활용하여 코드를 개선하거나 수정하는 것아다.
이를 통해 프로그래밍 학습 과정에서 코드 작성 오류를 줄이고, 더 효율적이고 정확하게 학습할 수 있다.

만들게 된 동기
정보 과목 수업을 진행하면서 직접 코드를 작성할 때 작은 문법 오류 하나 때문에 코드가 실행되지 않는 경우가 많았다. 또한 초보자가 스스로 코드를 개선하기 어려운 점을 발견했습니다.
그래서 학생이 작성한 코드를 자동으로 검사하고, ChatGPT를 통해 개선안을 제시할 수 있는 도구를 만들면 학습 효율을 높일 수 있을 것 같아 제작하게 되었다.

코드 설명

문법 검사 기능
사용자가 입력한 코드를 Python 내장 모듈 ast를 이용하여 문법적으로 오류가 있는지 확인한다.
오류가 있으면 오류 메시지를, 없으면 문법 오류 없음을 알려준다.

ChatGPT 자동 개선 기능
OpenAI의 GPT-4o-mini 모델을 사용하여 사용자가 입력한 코드를 분석하고, 오류를 수정하거나 코드 개선안을 제시합니다.
사용자는 버튼 클릭만으로 쉽게 수정된 코드를 받을 수 있다.

Streamlit 기반 웹 인터페이스

코드 입력, 문법 검사, ChatGPT 개선 제안 기능을 웹 페이지에서 직관적으로 이용할 수 있도록 구현했습니다.
버튼과 컬럼을 활용해 사용자가 쉽게 기능을 사용할 수 있도록 UI를 구성하였다.

기대 효과
-Python 코드를 배우는 학생들이 빠르게 오류를 확인하고 학습할 수 있다.
-초보자도 코드 개선 방법을 쉽게 배우고 적용할 수 있다.
-프로젝트 경험을 통해 AI 기술을 활용한 소프트웨어 개발 경험을 기록할 수 있다.
